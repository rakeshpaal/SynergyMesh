# Examples Directory

This directory contains usage examples for SynergyMesh.

## Structure

Examples are organized by feature or component:

```
examples/
├── provenance/     # SLSA provenance examples
├── contracts/      # Contract usage examples
├── automation/     # Automation workflow examples
└── integration/    # Integration examples
```

## Usage

Each example directory contains:

- README with setup instructions
- Sample code and configurations
- Expected outputs
