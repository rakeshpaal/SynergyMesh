# 🏢 貢獻者協議 / Contributor Agreement

感謝您有興趣為 SynergyMesh 專案貢獻力量！

Thank you for your interest in contributing to the SynergyMesh project!

---

## 📋 貢獻指南 / Contribution Guidelines

### 開始貢獻 / Getting Started

1. **Fork 倉庫** / Fork the repository
   ```bash
   git clone https://github.com/your-username/unmanned-island.git
   cd unmanned-island
   ```

2. **建立分支** / Create a feature branch
   ```bash
   git checkout -b feature/your-feature-name
   ```

3. **進行更改** / Make your changes
   ```bash
   # 編輯檔案
   # 添加測試
   # 更新文檔
   ```

4. **運行測試** / Run tests
   ```bash
   npm run lint
   npm test
   npm run test:coverage
   ```

5. **提交更改** / Commit your changes
   ```bash
   git commit -m "feat: 添加新功能"
   ```

6. **推送分支** / Push to your fork
   ```bash
   git push origin feature/your-feature-name
   ```

7. **建立 Pull Request** / Create a Pull Request

## ✅ 代碼質量標準 / Code Quality Standards

### TypeScript/JavaScript
- [ ] 遵循 ESLint 規則
- [ ] 無 TypeScript 錯誤
- [ ] 功能有相應的測試
- [ ] 代碼覆蓋率 > 80%
- [ ] 文件已更新

### Python
- [ ] 遵循 PEP 8 規範
- [ ] 使用 Type Hints
- [ ] 功能有相應的測試
- [ ] Docstring 已編寫
- [ ] 無 mypy 錯誤

---

## 📝 提交訊息格式 / Commit Message Format

遵循 Conventional Commits 格式：

```
<type>(<scope>): <subject>

<body>

<footer>
```

### 類型 / Types
- `feat`: 新功能
- `fix`: 修復錯誤
- `docs`: 文檔變更
- `style`: 代碼風格
- `refactor`: 重構
- `test`: 測試
- `chore`: 構建配置等
