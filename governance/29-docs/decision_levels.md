# L2-L4 決策模型

## 定義

| 層級 | 說明 | 決策者 | 審批 |
|------|------|--------|------|
| L4 | 戰略決策 | 人類主導 + AI 提供選項 | 多部門會審 |
| L3 | 戰術決策 | AI 主導 + 人類覆核 | 技術主管批准 |
| L2 | 執行決策 | AI 自主 + 事後審計 | 自動化流程 |

## 流程

1. 識別需求並標注風險
2. 決定最小必要層級
3. 如果需要提升層級，追加審批資料
4. 決策完成後寫入審計日誌

## 升降級規則

- 安全或金流相關：至少 L3
- 廣泛影響設定：L4
- 純文檔 / 格式化：L2
- 連續 3 次 rollback：自動升級一層

## 工具鏈

```bash
# 查詢決策
island-cli governance:decisions --level=L3

# 申請升級
island-cli governance:escalate --change-id=123
```
