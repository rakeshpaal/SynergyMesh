# API Overview

## Governance API

The Governance Framework provides RESTful APIs for managing governance operations.

### Base URL

```
http://localhost:8080/api/v1
```

### Authentication

API requires authentication token in header:

```
Authorization: Bearer <token>
```

See [API Endpoints](api-endpoints.md) for details.
