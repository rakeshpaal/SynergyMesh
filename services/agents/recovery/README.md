# Recovery Agent - Dr. Phoenix | 恢復代理 - 鳳凰博士

This directory contains Dr. Phoenix, the autonomous recovery expert agent.

## Files

- `phoenix_agent.py` - Main Phoenix Agent implementation
- `__init__.py` - Module initialization

## Usage

```bash
# Start Phoenix Agent
python phoenix_agent.py start

# Check status
python phoenix_agent.py status

# Stop Phoenix Agent
python phoenix_agent.py stop
```
