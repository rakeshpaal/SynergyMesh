# src

## 目錄職責

此目錄為 MachineNativeOps 的**核心源代碼**，包含所有業務邏輯、服務實作和系統引擎。

## 子目錄分類

### 核心引擎

| 子目錄 | 職責 |
|--------|------|
| `core/` | 核心引擎（AI 決策、編排器、Phase 4 系統、即時生成） |
| `runtime/` | 運行時環境（mind_matrix 認知處理器） |
| `ai/` | AI 代理與模型系統 |

### 自主與自動化

| 子目錄 | 職責 |
|--------|------|
| `autonomous/` | 自主系統框架（8 個已實現骨架） |
| `automation/` | 自動化引擎與架構設計工具 |

### 服務與 API

| 子目錄 | 職責 |
|--------|------|
| `services/` | 微服務實作 |
| `api/` | API 端點定義 |
| `server/` | 伺服器端實作 |
| `client/` | 客戶端實作 |

### 應用與前端

| 子目錄 | 職責 |
|--------|------|
| `apps/` | 應用程式 |
| `frontend/` | 前端代碼 |
| `web/` | Web 相關功能 |

### 治理與合約

| 子目錄 | 職責 |
|--------|------|
| `governance/` | 治理系統實作 |
| `contracts/` | 智能合約 |

### 工具與模板

| 子目錄 | 職責 |
|--------|------|
| `developer-tools/` | 開發者工具集 |
| `ci-tools/` | CI/CD 工具 |
| `templates/` | 代碼模板 |
| `utils/` | 通用工具函數 |
| `shared/` | 共享模組 |

### 其他

| 子目錄 | 職責 |
|--------|------|
| `models/` | 資料模型（database, dto, schemas） |
| `schemas/` | Schema 定義 |
| `mcp-servers/` | MCP 伺服器實作 |
| `bridges/` | 系統橋接模組 |
| `supply-chain/` | 供應鏈安全 |
| `next_gen/` | 下一代功能實驗區：用於原型設計、未來核心/自動化能力 PoC 與 A/B 測試，非正式生產代碼 |
| `代码圣殿/` | 高影響力/進階模組區：包含關鍵內核、進階編排器或安全敏感功能，需嚴格評審與治理流程 |

## 檔案說明

### demo_core.py
- **職責**：核心功能演示腳本
- **功能**：展示系統初始化、用戶認證等基本功能
- **依賴**：`core.new.core` 模組

### demo_instant_generation.py
- **職責**：即時生成功能演示
- **功能**：展示 Phase 4 即時代碼生成能力
- **依賴**：`core.instant_generation` 模組

## 職責分離說明

- **core/**：核心業務邏輯，不依賴具體框架
- **services/**：服務層，處理業務編排
- **api/**：接口層，處理 HTTP 請求/響應
- **models/**：資料層，定義資料結構

## 與其他目錄的關係

- **config/**：環境配置來源
- **tests/**：對應的測試代碼
- **docs/**：API 和架構文檔
