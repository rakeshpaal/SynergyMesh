# L0 治理政策

## 現狀

- L0（完全自動）在 Island AI 中預設停用
- 僅允許於低風險環境（Dev/Staging/背景作業）

## 啟用條件

1. 完整回滾機制（自動 + 手動）
2. 爆炸半徑受控（流量、範圍、時間）
3. 觀測到位（logs + metrics + traces）
4. 連續 100+ 次 Shadow Mode 無異常
5. 有 Kill Switch 與 Watchdog

## 禁區

- 金流轉帳
- IAM 根權限
- KMS 金鑰滾動
- 生產資料刪除

## 面向未來

- 擬定 L0 測試實驗場
- 建置自動審批/回滾模擬器
- 推動可證明安全的 L0 案例
