# frontend

## 目錄職責

此目錄為 MachineNativeOps 的**前端層**，包含 Web 使用者介面的實現。

## 子目錄說明

| 子目錄 | 職責 |
|--------|------|
| `ui/` | 使用者介面元件與頁面 |

## 目前狀態

此目錄為骨架結構，前端實現待補充。

## 設計原則

1. **元件化設計**：UI 元件可重用
2. **狀態管理**：集中管理應用程式狀態
3. **響應式設計**：支援多種裝置尺寸
4. **可存取性**：符合 WCAG 標準

## 依賴規則

**可以依賴**：
- `src/api/` - 透過 API 獲取資料
- `src/shared/types/` - 共用類型定義

**不可依賴**：
- `src/core/` - 前端不應直接依賴核心引擎
- `src/services/` - 前端不應直接依賴服務層
- `src/runtime/` - 前端不應依賴運行時

## 與其他目錄的關係

- **src/api/**：透過 API 層與後端通訊
- **src/shared/**：使用共用類型定義
