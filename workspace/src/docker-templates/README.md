# Docker Templates

This directory contains Docker template files for SynergyMesh.

## Contents

- `NODEJS_USER_SETUP.md` - Guide for setting up Node.js users in Docker
- `validate-dockerfiles.sh` - Script to validate Dockerfile configurations

## Usage

### Validating Dockerfiles

```bash
./validate-dockerfiles.sh
```
