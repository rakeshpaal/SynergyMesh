### Job Types

#### 1. Cron Jobs

```typescript
scheduler.schedule('backup', '0 2 * * *', async () => {
  await performBackup();
}, {
  timezone: 'UTC',
  priority: 'high',
  maxRetries: 3,
  timeout: 300000
});
```

**Use Cases:**

- Daily backups
- Weekly reports
- Monthly cleanup
- Periodic health checks

#### 2. One-Time Tasks

```typescript
scheduler.scheduleOnce('cleanup', new Date('2026-01-01T00:00:00Z'), async () => {
  await cleanupOldData();
}, {
  priority: 'normal',
  maxRetries: 1
});
```

**Use Cases:**

- Scheduled maintenance
- Future data processing
- Delayed notifications
- Time-based triggers

#### 3. Interval Tasks

```typescript
scheduler.scheduleInterval('health-check', 60000, async () => {
  await checkSystemHealth();
}, {
  priority: 'high',
  enabled: true
});
```

**Use Cases:**

- Health checks
- Metrics collection
- Status polling
- Real-time monitoring

## ğŸ“š Architecture Decision Records (ADRs)

### ADR-001: Frontend Framework Selection

**Decision**: React 18+ with TypeScript
**Reasoning**:

- Strong ecosystem and community
- TypeScript for type safety
- Performance optimizations (concurrent rendering)
- Excellent component libraries available

### ADR-002: Backend Framework Selection

**Decision**: Express.js with TypeScript
**Reasoning**:

- Lightweight and flexible
- Large middleware ecosystem
- Easy to integrate with Node.js tools
- TypeScript for maintainability

### ADR-003: Job Queue Technology

**Decision**: BullMQ with Redis
**Reasoning**:

- Production-proven reliability
- Distributed job processing
- Priority queues support
- Automatic retry mechanisms
- Real-time monitoring

### ADR-004: Authentication Strategy

**Decision**: JWT tokens with secure secret management
**Reasoning**:

- Stateless authentication
- Scalable across multiple instances
- Standard and widely supported
- Easy to integrate with external systems

### ADR-005: UI Component Library

**Decision**: shadcn/ui with TailwindCSS
**Reasoning**:

- Modern and customizable
- Accessible by default
- TypeScript support
- Excellent dark mode support
- Copy-paste components (no npm install)
