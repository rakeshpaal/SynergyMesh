# 身份與多租戶 - 自檢清單

## 設計前需求分析

- [ ] 認證需求是否確定（OAuth2、SAML、JWT 等）？
- [ ] 授權模型是否選定（RBAC、ABAC、ReBAC）？
- [ ] 租戶隔離級別是否定義（共享、半隔離、完全隔離）？
- [ ] 跨組織委派或聯邦需求是否已識別？

## 設計中架構設計

- [ ] 是否涵蓋所有使用者角色與權限等級？
- [ ] 是否考慮了跨租戶操作與權限衝突？
- [ ] 是否定義了權限提升流程與批准工作流？
- [ ] 租戶間資料隔離與存取控制是否已設計？

## 實現前驗證

- [ ] 認證機制是否通過安全評審？
- [ ] 租戶隔離策略是否已驗證（測試突破場景）？
- [ ] 故障恢復與備份程序是否已定義？
- [ ] 敏感操作（如權限變更）的審計日誌是否已設計？

## 上線前最終檢查

- [ ] 所有 API 端點是否已受授權保護？
- [ ] 是否進行了租戶隔離滲透測試？
- [ ] 是否配置了全面的審計日誌與監控？
- [ ] 是否進行了多租戶場景的失敗測試？
- [ ] 合規與安全團隊是否已批准上線？
