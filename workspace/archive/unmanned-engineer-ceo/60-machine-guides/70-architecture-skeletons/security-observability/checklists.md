# 安全與可觀測性 - AI 自檢清單

## 設計前必檢查

- [ ] 威脅模型是否已確定？
- [ ] 合規要求是否已識別？
- [ ] SLA 目標是否已定義？
- [ ] 監控預算是否已分配？

## 設計中必檢查

- [ ] 是否應用了零信任原則？
- [ ] 是否覆蓋了 CIA 三角（機密、完整、可用）？
- [ ] 是否設定了關鍵指標監控？
- [ ] 是否配置了告警規則？

## 實現前必檢查

- [ ] 安全設計是否通過評審？
- [ ] 監控基礎設施是否已準備？
- [ ] 告警渠道是否已配置？
- [ ] 事件響應程序是否已準備？

## 部署前必檢查

- [ ] 是否通過了安全掃描？
- [ ] 是否驗證了所有簽名？
- [ ] 是否配置了審計日誌？
- [ ] 是否進行了滲透測試？

## 驗證檢查

- [ ] 監控數據是否正確收集？
- [ ] 告警是否正確觸發？
- [ ] 安全控制是否有效運作？
- [ ] 合規報告是否已生成？
