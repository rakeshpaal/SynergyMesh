# 效能與可靠性 - 自檢清單

## 設計前需求分析

- [ ] 預期負載與成長預測是否已估計（用戶數、流量、資料量）？
- [ ] SLO 目標與可接受的故障時間是否已確定？
- [ ] 成本與效能的權衡是否已評估？
- [ ] 競爭對手或行業標準 SLO 是否已參考？

## 設計中架構設計

- [ ] 是否規劃了冗餘與多區域部署（故障轉移）？
- [ ] 是否考慮了所有可能的故障模式（組件、區域、級聯失敗）？
- [ ] 監控、告警與診斷設計是否已完成？
- [ ] 快速恢復與回滾策略是否已設計？

## 實現前測試

- [ ] 性能基準測試是否已進行並文件化？
- [ ] 容量規劃與自動擴展策略是否已驗證？
- [ ] 故障轉移是否已測試（手動與自動）？
- [ ] RTO/RPO 目標是否已在測試中驗證？

## 上線前最終檢查

- [ ] 是否進行了充分的負載與壓力測試？
- [ ] 是否進行了故障恢復演習與事後檢討？
- [ ] 災難復原計劃是否已驗證且團隊已培訓？
- [ ] SLO 儀表板與報告是否已就位？
- [ ] 預留容量與成本優化是否已平衡？
