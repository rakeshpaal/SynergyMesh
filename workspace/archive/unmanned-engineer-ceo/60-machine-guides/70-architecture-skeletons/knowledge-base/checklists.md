# 知識庫 - 自檢清單

## 設計前規劃

- [ ] 知識覆蓋範圍與邊界是否已明確？
- [ ] 預期查詢模式與用戶類型是否已識別？
- [ ] 知識更新頻率與維護流程是否已定義？
- [ ] 知識來源（文檔、代碼、ADR 等）是否已盤點？

## 設計中架構設計

- [ ] 知識組織架構是否採用語義而非檔案系統？
- [ ] 實體間的關係與依賴是否已定義？
- [ ] 版本控制與知識演進追蹤是否已規劃？
- [ ] 搜尋與相關性排名算法是否已設計？

## 實現前連接

- [ ] 知識查詢引擎是否已實現並測試？
- [ ] 所有知識源是否已連接與同步？
- [ ] 自動更新機制是否已配置（如代碼變更時更新）？
- [ ] 知識完整性檢查是否已自動化？

## 上線前最終檢查

- [ ] 知識準確性與代碼同步是否已驗證？
- [ ] 查詢性能是否滿足需求（< 1 秒）？
- [ ] AI 上下文與相關文檔推薦是否有效？
- [ ] 是否進行了大規模查詢負載測試？
- [ ] 知識維護與更新責任是否已指派？
