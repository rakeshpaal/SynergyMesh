# 成本管理 - 自檢清單

## 設計前預估階段

- [ ] 預期成本是否已估計（含基礎設施、人力、第三方服務）？
- [ ] 預算是否已獲得財務批准且有成本中心分配？
- [ ] 成本目標與 ROI 預期是否已明確定義？
- [ ] 是否已識別成本主要驅動因素（計算、存儲、網絡等）？

## 設計與架構階段

- [ ] 是否考慮了多個成本優化方案並進行比較？
- [ ] 是否選擇了成本效益的技術棧與架構模式？
- [ ] 是否規劃了成本監控與告警策略？
- [ ] 成本與效能/可靠性的權衡是否已文件化？

## 實現與部署前

- [ ] 資源配置是否已優化（正確的實例類型、預留容量）？
- [ ] 成本監控與追蹤是否已啟用（按服務、按成本中心）？
- [ ] 預算警報與成本異常偵測是否已配置？
- [ ] 是否準備了成本優化建議與補救流程？

## 上線前最終檢查

- [ ] 預期月度成本是否合理且在預算範圍內？
- [ ] 是否進行了成本演習與場景分析（如流量激增）？
- [ ] 團隊是否已培訓成本監控與優化流程？
- [ ] 是否有成本所有者與審查週期（每月成本審查會議）？
- [ ] 成本數據與帳單是否已驗證與對帳？
