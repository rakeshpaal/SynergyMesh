# Island AI 產品概覽

## 定位

- 企業級 AI 工程平台
- 統一控制層，涵蓋研發、治理、自治四大領域
- 面向多語言、多團隊、多雲環境

## 價值主張

1. **自主演化**：Agent 生態持續自學習並優化決策
2. **全棧整合**：從 Rust 基礎設施到 TypeScript 應用層的完整鏈路
3. **治理先行**：L2-L4 分層審批確保安全可控
4. **自我感知**：活體知識庫與健康診斷維持系統韌性

## MVP 三支柱

| 支柱 | 說明 |
|------|------|
| 自動化 | 自動檢測、自動修復、自動派工 |
| 治理 | Schema Pipeline、SLSA、策略閘門 |
| 自主 | 五骨架自主框架、無人機/自駕整合 |

## 成熟度路線

- **v0.1**：品牌遷移、Runtime 骨架
- **v0.5**：Agent 生態 + 治理框架
- **v1.0**：多語言島嶼 + island-cli + 自主系統
- **v2.0**：自主演化 + 市場外部插件
