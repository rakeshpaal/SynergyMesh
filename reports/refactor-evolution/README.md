# Refactor & Evolution Reports

# 重構與演化報告

This directory contains generated reports from the Auto Refactor & Evolution System.

此目錄包含自動重構與演化系統生成的報告。

## Directory Structure / 目錄結構

```
refactor-evolution/
├── README.md                    # This file
├── .gitignore                   # Ignore generated reports
├── analysis_*.yaml              # Analysis reports (generated)
├── workflow_report_*.yaml       # Workflow execution reports (generated)
├── logs/                        # Execution logs
│   └── .gitkeep
└── plans/                       # Execution plans
    └── .gitkeep
```

## Generated Files / 生成的文件

### Analysis Reports

- **Pattern**: `analysis_YYYYMMDD_HHMMSS.yaml`
- **Contains**: Codebase structure analysis, identified issues, recommendations
- **Generated by**: Analysis phase

### Workflow Reports

- **Pattern**: `workflow_report_YYYYMMDD_HHMMSS.yaml`
- **Contains**: Complete workflow execution summary, metrics, results
- **Generated by**: Workflow completion

### Execution Plans

- **Pattern**: `plans/plan_YYYYMMDD_HHMMSS.yaml`
- **Contains**: Refactoring execution plans, prioritized actions
- **Generated by**: Planning phase

### Logs

- **Location**: `logs/`
- **Contains**: Detailed execution logs
- **Format**: Various (text, YAML)

## Note / 注意

Generated reports are automatically ignored by git (see `.gitignore`).
Only the directory structure is tracked.

生成的報告會自動被 git 忽略（見 `.gitignore`）。
僅追蹤目錄結構。

## Cleanup / 清理

To clean old reports:

```bash
# Remove reports older than 7 days
find reports/refactor-evolution/ -name "*.yaml" -mtime +7 -delete

# Remove all generated reports
rm -f reports/refactor-evolution/analysis_*.yaml
rm -f reports/refactor-evolution/workflow_report_*.yaml
rm -rf reports/refactor-evolution/logs/*
rm -rf reports/refactor-evolution/plans/*
```
